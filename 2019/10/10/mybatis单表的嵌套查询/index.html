数据库查询思路:

首先查询父栏目 例如: 体育新闻,

select * from newslabel where id = 1

找到体育新闻, 然后用体育新闻的id查询

select * from newslabel where pid = #{id}

取结果集的id再次作为条件查询

select * from neswslabel where pid = #{id}

反复这样, 直到再也不到结果 结束
